CYOA Dark Fantasy Project

Content & Context

This new project conducted is much more complex and complicated than the movie project. The prompt for this project was to make a game of some sort, learning CYOA, or Choose Your Own Adventure. The topic decided by me and my partner was Dark Fantasy. Practicing links and moving from file to file using links in our IDE's.

Reflection & Takeaways

The Project being no more than less than one week, me and my partner planned and split the story into two parts. One of us did one part, and the other, another. Doing this made workflow much easier and code to flow. Upon nearing our end of the project once it was completed, we ha some adversaries. One being the names of .md's having capital letters or periods, which messed up the code both after pulling, or when the partner had to edit pushes. I also helped fixed their issues they were having, and with time, revised work several times. Other than fixing some links, and file names, not much more of an issue occured from there on out.

Next steps

From this point on, if I had more time on the project, I'd want to add hmtl and images. Or make a real game based on these events as an easter egg. As well as gain more knowledge in my IDE and the coding techniques.
